Introduction
------------

This Python script generates mazes as Adv. Dupe files.

Known Issues
------------

- The maze algorithm implemented doesn't create very complicated mazes.
- The generator doesn't try to coalesce walls together to reduce the
  number of props used.

Requirements
------------

- Python >= 2.5
- pyAdvDupe module:
  <http://github.com/sk89q/pyadvdupe>

Usage
-----

./maze.py
