Introduction
------------

Montezuma's Wheel is a synchronized music-oriented ride.

Installation
------------

Drop the folder in the "e2files" folder into an "e2files" folder in your
game's "data" directory. If the e2files folder doesn't exist in data/,
then create it.

Spawn the advanced duplicator file.

You can also create an instance of the "ride_origin" E2 file to move the
origin of the ride. The scripted rides that come with Montezuma's Wheel
require up to 6000 units of space (3000 radius).
